# Project Charter

## Vision
Build a department-owned curriculum database that serves as the single source of truth for all Middlesex mathematics courses.

The curriculum—not any textbook—is authoritative. OpenStax, workbooks, assessments, Desmos activities, videos, and AI tutoring are resources attached to the curriculum.

## Goals
- Consistent learning objectives
- Reusable skills library
- Horizontal alignment
- Teacher planning
- Student learning tools
- AI-assisted curriculum development

## Guiding Principle
Learning Objectives and Skills are permanent.
Resources are replaceable.
