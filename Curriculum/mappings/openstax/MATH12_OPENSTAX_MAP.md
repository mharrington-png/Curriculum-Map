# Math 12 OpenStax Mapping

This draft translates the existing Math 12 course map into an objective-centered resource crosswalk. OpenStax sections are replaceable resources; the Math 12 objectives remain authoritative.

## Resource Record

- **Resource:** OpenStax *Intermediate Algebra 2e*
- **Authors:** Lynn Marecek and Andrea Honeycutt Mathis
- **Published:** 2020
- **Mapping level:** Textbook section to learning objective
- **Current scope:** Section-level alignment only; individual examples and exercises are not yet mapped

## Alignment Labels

- **Direct:** The section explicitly teaches the stated learning objective.
- **Partial:** The section supports part of the objective, but supplementation or careful selection is required.
- **Supporting:** The section supplies prerequisite or adjacent material rather than teaching the full objective.
- **Gap:** No section in the currently approved range directly teaches the objective.

## Objective Gaps and Boundaries Identified at Section Level

| Objective | Status | Section-level finding |
|---|---|---|
| M12-INE-005 | Gap | Section 2.7 is an absolute-value **inequality** section. A separate resource is needed for solving absolute-value equations. |
| M12-LIN-009 | Partial | Section 3.6 supports graph-based domain and range, but contextual restrictions require additional material. |
| M12-EXT-002 | Partial | Section 3.4 teaches one linear inequality in two variables; a system-of-inequalities resource is still needed. |
| M12-EXT-003 | Partial | Section 6.1 provides the approved factoring support for this extension objective; select applicable examples and supplement only if additional trinomial depth is desired. |
| M12-EXT-004 | Partial | Discrete-versus-continuous classification is not sufficiently explicit in the mapped function sections. |
