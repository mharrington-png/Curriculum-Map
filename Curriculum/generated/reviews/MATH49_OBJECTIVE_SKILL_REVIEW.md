# M49 Objective-to-Skill Mapping Review

This teacher-facing view combines each learning objective with the supporting skills currently mapped to it. The YAML files remain the structured source; this document is generated for review.

## How to Read the Mapping

- **Prerequisite:** Students should bring this skill into the objective.
- **Required:** The skill is inherent in successfully meeting the objective.
- **Method-dependent:** The skill is needed for one accepted method, but not every method.
- **Introduce:** First sustained instruction in this course sequence.
- **Reinforce:** Revisit at comparable depth.
- **Deepen:** Add complexity, interpretation, representation, or independence.
- **Apply:** Use an established skill as a tool inside a new objective.
