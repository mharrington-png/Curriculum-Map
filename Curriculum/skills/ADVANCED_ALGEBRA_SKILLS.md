# Advanced Algebra Supporting Skills

This is the initial reusable skill library extracted from Math 31. Foundational and intermediate skills should be referenced from the existing skill files rather than duplicated.

## Function Structure Skills

- **SK-FUNC-NOTATION-SOLVE:** Use function notation to express and solve an input-output question.
- **SK-FUNC-INDEPENDENT-DEPENDENT:** Identify independent and dependent variables.
- **SK-FUNC-INTERVAL-BEHAVIOR:** Describe increasing or decreasing behavior over intervals.
- **SK-FUNC-AROC-TOTAL-CHANGE:** Use average rate of change and interval length to calculate total change.
- **SK-FUNC-AROC-UNITS:** Determine rate units from input- and output-axis units.
- **SK-FUNC-PIECEWISE-EVALUATE:** Select the correct rule of a piecewise function for a given input.
- **SK-FUNC-PIECEWISE-GRAPH:** Graph each rule on its stated domain with correct endpoints.
- **SK-FUNC-PIECEWISE-WRITE:** Translate a segmented graph into a piecewise formula.
- **SK-FUNC-DOMAIN-EVEN-ROOT:** Require an even-root radicand to be nonnegative when determining domain.
- **SK-FUNC-TRANSFORM-HORIZONTAL:** Interpret transformations applied to function inputs.
- **SK-FUNC-TRANSFORM-VERTICAL:** Interpret transformations applied to function outputs.
- **SK-FUNC-TRANSFORM-SCALE:** Distinguish stretches from compressions and horizontal effects from vertical effects.
- **SK-FUNC-COMPOSE-FORMULA:** Substitute one function formula into another.
- **SK-FUNC-COMPOSE-TABLE:** Follow intermediate values to compose functions from tables.
- **SK-FUNC-INVERSE-SWAP:** Exchange input and output values when interpreting an inverse.
- **SK-FUNC-INVERSE-ALGEBRA:** Exchange variables and solve to obtain an inverse formula.
- **SK-FUNC-INVERSE-DOMAIN-RANGE:** Exchange domain and range between a function and its inverse.
- **SK-FUNC-INVERSE-UNITS:** Reverse input and output units when interpreting an inverse.
- **SK-FUNC-INVERSE-HLT:** Apply the horizontal line test for invertibility.
- **SK-FUNC-CONCAVITY-AROC:** Use successive average rates of change to classify concavity.

## Exponential Skills

- **SK-EXP-DIFFERENCE-RATIO:** Use constant differences or ratios to distinguish linear and exponential patterns.
- **SK-EXP-PERCENT-FACTOR:** Convert a percent increase or decrease to a growth or decay factor.
- **SK-EXP-MODEL-TABLE:** Determine an exponential model from tabular values.
- **SK-EXP-MODEL-TWO-POINTS:** Solve for an exponential model using two points.
- **SK-EXP-MODEL-CONTEXT:** Identify initial amount, rate, and time interval in context.
- **SK-EXP-COMPOUND-PERIODIC:** Substitute correctly into a periodic-compounding model.
- **SK-EXP-COMPOUND-CONTINUOUS:** Substitute correctly into a continuous-compounding model.
- **SK-EXP-RATE-CONVERT:** Convert between nominal and effective rate descriptions.
- **SK-EXP-DISCRETE-CONTINUOUS:** Convert between equivalent discrete- and continuous-growth forms.
- **SK-EXP-COMPARE-GROWTH:** Compare exponential growth rates expressed in different forms.
- **SK-EXP-ASYMPTOTE:** Identify horizontal asymptotic behavior of an exponential graph.

## Logarithmic Skills

- **SK-LOG-DEFINITION:** Interpret a logarithm as an exponent.
- **SK-LOG-PRODUCT:** Apply the logarithm product property in either direction.
- **SK-LOG-QUOTIENT:** Apply the logarithm quotient property in either direction.
- **SK-LOG-POWER:** Apply the logarithm power property in either direction.
- **SK-LOG-SOLVE-EXP:** Isolate an exponential expression and apply a logarithm.
- **SK-LOG-SOLVE-LOG:** Combine or isolate logarithms before converting to exponential form.
- **SK-LOG-DOMAIN-CHECK:** Reject solutions that make a logarithm argument nonpositive.
- **SK-LOG-DOUBLING-HALF:** Translate a doubling-time or half-life question into an exponential equation.
- **SK-LOG-ASYMPTOTE:** Identify vertical asymptotic behavior of a logarithmic graph.

## Representation and Interpretation Skills

- **SK-REP-FORMULA-GRAPH:** Connect parameters in a formula to features of its graph.
- **SK-REP-TABLE-GRAPH:** Use tabular patterns to predict graph shape and behavior.
- **SK-REP-UNITS:** Attach and interpret units for rates, inputs, and outputs.
- **SK-MODEL-PARAMETER-INTERPRET:** Explain a model parameter in the language of its context.
- **SK-MODEL-VALID-DOMAIN:** Restrict a model to meaningful contextual inputs.
- **SK-LOG-SCALE-INTERPRET:** Interpret equal intervals on a logarithmic scale as equal multiplicative changes.
