# Retired Skill Aliases

This file preserves the migration path from broad or duplicate draft skill IDs to canonical, more precise records. Retired IDs must not be assigned to new objectives or resources.

|---|---|---|

## Governance Rule

When two skill records overlap, retain the smallest observable skill that is reusable across courses. Use multiple canonical skill IDs when a broad classroom task depends on several supporting techniques.
